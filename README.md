## 在本地运行

Clone 这个 project

```bash
  git clone https://github.com/MeteorCodes/Meteor_AI.git
```

前往项目目录

```bash
  cd Meteor_A
```

安装依赖

```bash
  pip install flask openai
```

启动服务器

```bash
  python demo.py
```
